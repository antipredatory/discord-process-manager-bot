# Installation
1. **Clone the Repo**
```bash
git clone https://github.com/antipredatory/discord-process-manager-bot
```
2. install dependencies
```bash
pip install -r requirements.txt
```
3. Add your bot token
```py
bot.run('token_here')
```
4. Run 
```bash
py main.py
```
